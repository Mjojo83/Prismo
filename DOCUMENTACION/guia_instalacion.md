# Guía de Instalación

1. **Python**: Instalar Python 3.10+
2. **Dependencias**:
   ```bash
   pip install -r CEREBRO/requirements.txt
   ```
3. **Godot**:
   - Descargar Godot 4 (Standard version es suficiente).
   - Importar el proyecto desde la carpeta `CUERPO/project.godot`.
   
## Ejecución
1. Iniciar el Cerebro:
   ```bash
   python CEREBRO/main.py
   ```
2. Iniciar el Cuerpo:
   - Ejecutar desde Godot Editor o exportar binario.
